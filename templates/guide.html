Website Guide

Follow this brief guide to learn everything about this website!

For more information feel free to contact us directly!

EMPLOYEES PAGE

This page page allows you to input the information from each worker in your firm, which will automatically be registered in the database. Assign an ID to each employee and then use it to create connections with the information from the other pages!

SKILLS PAGE

This page allows you to input both the hard and soft skills for each employee (use the personal ID to assign them to the right employee).

JOB OPENINGS PAGE

This page allows you to input the jobs available in the firm and the skills required for each. The system will then indicate you the right employees for the job based on the information you previously provided.

PROJECTS PAGE

Similarly to the Job Openings page, this page allows you to input the projects available in the firm and the skills required for each. The system will then indicate you the right employees for the project based on the information you provided previously. In addition it will show you the current projects and the archive.

TRAININGS PAGE

This page will contain the courses list and the upcoming courses, which can be allocated to the employees which need them due to a lack of a certain skill. An archive of the courses allows you to keep track of the courses been completed.
